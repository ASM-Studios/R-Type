GUI::WindowManager::WindowManager ( )

Constructs a WindowManager object.

Initializes the WindowManager with the default configuration. Loads the configuration from the specified config file. Initializes the window with the specified width and height. Initializes the music manager with the main theme music. Initializes the player entity.

Exceptions
GuiExceptionif there is an error reading the config file.
